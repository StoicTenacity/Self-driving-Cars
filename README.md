# Self-driving Cars

**Please note that it take several attempts to train the cars to dodge around traffic successfully, refresh the page and try again!**

[Demo page](https://stoictenacity.github.io/Self-driving-Cars/)

Features:
* [X] upgraded basic traffic system to allow infinite traffic that allows users to continuously test a neural network
* [X] GUI for variable configurations(number of cars and mutation rate)
* [X] Pause/Resume functionality
* [X] Timer
* [X] Leaderboard
* [X] Remaining number of cars tracker
